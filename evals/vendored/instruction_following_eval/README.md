# Vendored IFEval Reference Implementation

This directory contains vendored code from the official IFEval (Instruction-Following Evaluation) implementation.

## Source

- **Repository**: https://github.com/google-research/google-research/tree/master/instruction_following_eval
- **License**: Apache License 2.0

## Purpose

These files are vendored to ensure our evaluation results match the official IFEval leaderboard exactly. We use the reference implementation's evaluation logic without modifications.
